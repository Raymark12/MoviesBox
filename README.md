# MovieBox

Movie recommendation web app built with React Router, TypeScript, and Chakra UI.

## Stack

- React Router v7 (framework mode, SPA)
- TypeScript
- Chakra UI
- Vite

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## Scripts

- `npm run dev` - development server
- `npm run build` - production build
- `npm run typecheck` - type checking
